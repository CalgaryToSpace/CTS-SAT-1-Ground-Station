# gnu_radio_usrp_b210

CTS-SAT-1 GNU Radio connection to the USRP B210.

## `usrp_b210_gnu_radio/`

The USRP B210 SDR is a high-quality SDR, and is the primary radio used for CTS-SAT-1 operations.

## `icom_ic_9700_gnu_radio/`

The ICOM IC-9700 radio lacks the ability to uplink at 9600 baud.

This flowgraph is entirely unused, and does not work.

## `supporting_demo_scripts/`

Demo scripts for interfacing with the GNU Radio flowgraphs.
